Heroku

Heroku is used to host projects that require a server, just as projects built using a backend framework. Heroku uses a ephemeral file system and everytime the worker restarts, approximately in 12 hours, all user uploaded content is deleted. To get around this, store media files on a external sorce such as Amazon Web Services.
